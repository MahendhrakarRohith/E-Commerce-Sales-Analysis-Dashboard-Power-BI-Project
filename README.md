📊 E-Commerce Sales Analysis Dashboard | Power BI Project

my latest Power BI project focused on analyzing e-commerce sales performance and customer trends.

🔹 Tools Used: Power BI, Power Query, DAX, Excel

🔹 Key Features:
✅ Interactive dashboard for sales tracking and analysis
✅ Dynamic filtering using slicers and drill-down functionality
✅ Data modeling with table relationships and calculations
✅ KPI tracking for revenue, profit, quantity sold, and customer performance
✅ Multiple visualizations including bar charts, line charts, donut charts, maps, and scatter plots

🔹 Project Learnings:
• Data cleaning and transformation
• Data modeling and relationship management
• Creating calculated measures using DAX
• Building interactive dashboards for business decision-making
• Visual storytelling through data

📈 This project helped me strengthen my data analytics and business intelligence skills while gaining hands-on experience with real-world sales data.
